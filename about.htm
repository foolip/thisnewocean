About This eBook

This EPUB version of This New Ocean is based on the HTML version published by the NASA History Office. Many small changes and improvements have been made:

  • Hundreds of typos have been found and fixed by comparing the text to an independent scan of the book from the Internet Archive.
  • Missing illustrations have been added using high-resolution versions published by NASA, the Internet Archive scans, and other sources.
  • Some illustrations have been replaced by alternatives, where they were judged superior to those in the book.
  • Illustrations have been reorganized to better fit the EPUB medium.

A complete list of changes and the tools used can be found in the Git repository. This eBook was generated from commit REVISION.
